# 边界值清单

## 输入框通用边界

| 字段类型 | 边界值检查项 |
|---------|-------------|
| 文本 | 空字符串、单字符、maxlength-1、maxlength、maxlength+1、超长文本(1000+)、特殊字符(<>\"'&)、emoji、纯空格、首尾空格 |
| 数字 | 负数、0、1、max-1、max、max+1、小数、科学计数法、非数字字符 |
| 邮箱 | 无@、无域名、超长邮箱、中文邮箱、特殊字符 |
| 手机号 | 少于11位、多于11位、非数字、非1开头 |
| 密码 | 纯数字、纯字母、无特殊字符、少于最低位数 |
| 日期 | 过去日期、未来日期、当天、2月29日（非闰年）、非法格式 |
| 下拉 | 不选（默认值）、第一项、最后一项 |
| 金额 | 0、负数、极大值、小数位数超限 |

## 列表/表格通用边界

- [ ] 0 条数据 → 空状态是否有引导？
- [ ] 1 条数据 → 边界显示是否正常？
- [ ] 一页满数据 → 分页是否正确？
- [ ] 大量数据（万级）→ 性能/虚拟滚动是否需要？
- [ ] 搜索无结果 → 空状态是否有"无结果"提示？
- [ ] 筛选后数据为空 → 是否有"清空筛选"入口？

## 上传文件通用边界

- [ ] 未选择文件 → 是否有提示？
- [ ] 超大小文件 → 是否在前端拦截？
- [ ] 不允许的格式 → 是否在前端提示？
- [ ] 文件名超长 → 是否截断或拒绝？
- [ ] 同时上传多个文件 → 是否限制数量？
- [ ] 上传过程中断网 → 是否支持断点续传？

## 分页通用边界

- [ ] 首页 → 上一页按钮是否禁用？
- [ ] 末页 → 下一页按钮是否禁用？
- [ ] 跳转到超出范围的页码 → 是否拦截？
- [ ] 切换每页条数后 → 是否正确重新分页？
